# CODEMORE_WEEK2_TASK5

Design a 'BankAccount' class with private attributes for account number and balance. Implement public getter and setter methods to access and modify the balance, with validation to prevent negative balances. This exercise will help you understand the principles of encapsulation and data hiding.
